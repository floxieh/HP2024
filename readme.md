
# ⚡️ HP Blancheporte ⚡️

Nouveaux encarts HP édition 2024 👩‍💻

## 🚀 Lancer le projet 

Cloner la HP (Depuis le terminal)

```bash
git clone https://github.com/blancheporte/HP2024.git
```

Se rendre dans le dossier de la HP

```bash
cd HP2024
```

Installer les dépendances

```bash
npm install
```

Démarrer le serveur

```bash
npm run dev
```

Exporter la HP

```bash
npm run build
